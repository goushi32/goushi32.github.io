PHP

一、HTML 基础骨架

表单上传核心标签

  • <form>：表单根标签，包裹输入框、文件、提交项
  • action：提交数据跳转/处理的 PHP 文件
  • method="post"：上传文件必须 POST，GET 无法传输文件
  • enctype="multipart/form-data"：二进制传输文件，上传必备属性

二、PHP 常用输出函数

函数 作用 示例
echo 普通内容输出 echo "hello";
print_r 打印数组/对象 print_r($arr);
var_dump 调试：输出值+类型 var_dump($var);
var_export 输出可执行 PHP 代码 var_export($arr);

三、PHP 字符串处理函数

函数 作用
strlen 获取字符串长度
substr 截取字符串
str_replace 字符串替换
strpos 查找字符首次下标
trim 去除首尾空格/空白符
explode 字符串分割为数组
implode/join 数组拼接为字符串
strtoupper 字母转大写
strtolower 字母转小写
htmlspecialchars HTML 转义，防御 XSS
strip_tags 清除 HTML/PHP 标签

四、PHP 数组

基础

  • array()：创建数组
  • count()：统计数组元素个数

数组遍历 foreach

PHP 专属数组循环，适配索引数组、关联数组

示例：

  • $x：当前元素键名
  • $x_value：当前元素值

数组常用函数

  • array_push：尾部追加元素
  • array_pop：删除尾部元素
  • array_merge：合并多个数组
  • in_array：检测值是否存在数组
  • array_keys：获取全部键名
  • array_values：获取全部值
  • array_unique：数组去重

数组排序

  • sort()：索引数组升序
  • rsort()：索引数组降序
  • asort()：按值升序（关联数组）
  • ksort()：按键升序
  • arsort()：按值降序
  • krsort()：按键降序

五、PHP 超级全局变量

全局作用域直接使用，无需定义

  • $GLOBALS：存放所有全局变量的数组
  • $_SERVER：服务器、路径、请求头信息
  • $_REQUEST：接收 GET / POST / COOKIE 数据
  • $_POST：接收 method="post" 表单数据
  • $_GET：接收 URL 参数 / get 表单

六、PHP 循环结构

  • while：条件成立循环执行
  • do...while：先执行一次，再判断条件
  • for：固定次数循环
  • foreach：专门遍历数组

八、PHP 魔术常量

  • __LINE__：当前代码行号
  • __FILE__：文件完整绝对路径
  • __DIR__：文件所在目录
  • __FUNCTION__：当前函数名
  • __CLASS__：当前类名
  • __METHOD__：类+方法名
  • __NAMESPACE__：当前命名空间
  • __TRAIT__：当前 Trait 名称

PHP 不支持多继承，使用 Trait 实现多类代码复用

  • namespace 必须为文件第一行有效代码（declare 除外）

基础示例

表单安全处理

  • trim()：去除首尾空格、制表符、换行
  • stripslashes()：清理多余反斜杠
  • htmlspecialchars()：转义特殊字符，防御 XSS

十三、PHP 文件引入

  • include：引入文件，报错警告，继续运行
  • include_once：只引入一次，防止重复加载
  • require：引入文件，致命错误，终止程序
  • require_once：仅加载一次+致命错误

文件操作函数

  • fopen()：打开文件
  • fclose()：关闭文件
  • fgetc()：逐字符读取文件

文件打开模式

  • r：只读，文件开头
  • r+：读写，文件开头
  • w：清空写入，不存在则创建
  • w+：清空读写
  • a：尾部追加写入
  • a+：追加读写
  • x：新建写入，存在则报错
  • x+：新建读写

十四、PHP Cookie

  • 作用：本地存储用户临时数据
  • 限制：setcookie() 必须写在 <html> 之前

十五、PHP Session

  • 作用：服务端保存用户会话
  • 开启：session_start() 必须在 <html> 前
  • 销毁变量：unset($_SESSION['xxx'])
  • 销毁会话：session_destroy()

十六、PHP 邮件发送

邮件安全过滤

  • FILTER_SANITIZE_EMAIL：清理非法邮箱字符
  • FILTER_VALIDATE_EMAIL：严格验证邮箱格式

十七、PHP 错误处理

自定义错误级别

  • E_USER_ERROR：致命错误
  • E_USER_WARNING：运行警告
  • E_USER_NOTICE：普通提示
