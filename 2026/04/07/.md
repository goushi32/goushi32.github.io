总结及学习内容：  
相对路径和绝对路径的区分，相对路径更加简便是起始路径/末尾路径，绝对路径是沿着路径到最后。  
ls -a -l -h 路径 。展示文件夹的目录。-a（all）能展示出一些隐藏的文件文件夹。-l以列的形式展示目录同时会有其他的信息左到右权限、所属用户、所属用户组等。
cd 路径.更改目前所属的路径。  
pwd 路径。展示所属路径。  
mkdir  路径文件夹名。创建新的文件夹。-p可以将所属路径上的未有的文件夹全部创建。  
touch  路径文件名。创建新文件。  
cat  路径文件名。查看文件内容。  
more  查看文件内容内容过多时可以翻页。通过空格进行翻页q退出。  
cp -r 被复制文件 复制到的路径。复制文件文件夹。-r文件夹及其文件内容。  
mv -r 文件文件夹名 要去的地方。  
rm -r -f  文件文件夹名 。可以多个。  
which 查看命令所在文件。  
find 起始路径 -name "被查文件名"。  
find 起始路径 -size+（&-）数字kMG。  
grep -n 关键字 路径。在路径中查找关键字的内容，-n显示行数。重定向符>左侧内容覆盖到右侧文件中>>左侧内容追加到右侧文件中。  
wc -c -m -l -w 路径。-c显示byte数量-m显示字符数量-l行数-w显示单词数。  
echo 输出内容。在命令行显示内容。复杂内容用"".反引号``中内容当作命令。  
tail -f -num（确切数字）路径。查看文件尾部内容。-f持续追踪。-num查看行数不加默认10行。  
vi/(vim) 文本编译器。  
通配符*test以test结尾的*test*含有的test*以他开头的。  
管道符|左侧输出作为右侧输入。
